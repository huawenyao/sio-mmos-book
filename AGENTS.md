## CONVENTIONS
- **无构建系统**：直接编辑HTML/CSS/JS，浏览器即运行
- **index.html 特殊**：唯一全内联样式+脚本的页面（book-show.html 和 reader.html 用外部文件）
- **中文文件名**：Markdown 源文件含中文与空格，reader.js 中 `chapterMapping` 硬编码这些路径
- **CSS变量命名**：`--bg-primary`, `--text-primary`, `--gradient-1` 等语义化命名
- **Canvas动画**：所有可视化（粒子、知识图谱、模态框图示）均用原生Canvas 2D，无第三方库
- **marked.min.js vendored**：本地拷贝，非npm引入

## ANTI-PATTERNS (THIS PROJECT)
- **勿编辑 `marked.min.js`** — 标有 "DO NOT EDIT THIS FILE"
- **勿在 reader.js 外修改章节路径** — `chapterMapping` 是唯一数据源
- **勿删除 generate.js** — 用于重新生成 index.html 内联样式（尽管目前可手动维护）

## UNIQUE STYLES
- `data-theme` 属性切换：reader 支持 `light` / `dark` / `sepia` 三主题
- 本地存储持久化：阅读进度、字体大小、主题、书签均存 `localStorage`
- 纯静态部署：任意HTTP服务器即可，`fetch()` 加载同级目录 `../SSIO-BOOK/` 的Markdown

## COMMANDS

```bash
open http://localhost:8000/book-show/
```

## NOTES
- Markdown文件使用中文命名且含空格，移动/重命名需同步更新 `reader.js` 的 `chapterMapping`
- `book-show.js` 同时服务于 `book-show.html`（粒子动画、知识图谱、Tab切换、模态框）
- `reader.js` 的 `chapterMapping` 路径使用 `../SSIO-BOOK/` 相对路径，部署时目录结构不可变
- 无 Service Worker / PWA 功能（注释中标记为"待实现"）
