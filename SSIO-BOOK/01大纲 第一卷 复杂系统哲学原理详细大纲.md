## 第二章：涌现与层展 —— 复杂性的形而上学基础

- 2.1.3 涌现的类型学：简单涌现、复杂涌现、自创生

**涌现的分类框架**：
```
涌现类型谱系：

弱涌现 ──────────────────────────────→ 强涌现
(可预测/可推导)                    (不可预测/不可推导)
    │                                    │
    ├─ 简单涌现                          ├─ 强向下因果
    ├─ 模式涌现                          ├─ 新因果能力
    └─ 功能性涌现                        └─ 本体论涌现
```

### 2.2 层展论与多层级实在
- 2.2.1 层展论的物理主义基础
- 2.2.2 层级之间的因果关系：向上因果与向下因果
- 2.2.3 层级自主性与层级耦合

**层展论的核心命题**：
1. **层级不可还原性**：高层级规律不能从低层级完全推导
2. **层级自主性**：每个层级具有自身的解释资源和因果效力
3. **层级间的双向因果**：既有向上因果（微观→宏观），也有向下因果（宏观→微观）

**关键概念：向下因果**
```
向下因果的机制：

系统层级约束 → 改变组分行为 → 组分状态改变 → 系统状态改变
      ↑_________________________________________________|
                        （反馈循环）

示例：
- 自然选择（种群层级 → 基因频率改变）
- 意识控制（心理层级 → 神经激活模式改变）
- 市场机制（经济系统层级 → 个体决策改变）
```

### 2.3 自创生与操作闭合
- 2.3.1 马图拉纳与瓦雷拉：自创生理论
- 2.3.2 操作闭合与结构耦合
- 2.3.3 从生命到认知：自创生认知论

**自创生的定义**：
```
自创生系统 = 通过其自身的运作持续再生产其自身的系统

关键特征：
1. 操作闭合：系统的运作形成闭合的网络
2. 结构决定：系统的结构决定其可能的交互
3. 结构耦合：系统与环境通过反复交互相互构成
```

### 2.4 涌现本体论的哲学意涵
- 2.4.1 涌现与因果性：强涌现是否违反物理因果闭合？
- 2.4.2 涌现与自由意志：涌现能否为自由意志提供本体论基础？
- 2.4.3 涌现与心物问题：意识是否是大脑的涌现属性？

**哲学争论的核心**：
```
强涌现的因果效力问题：

如果强涌现是真实的（即涌现属性具有不可还原的因果效力），
那么它似乎违反了物理因果闭合原则。

可能的解决方案：
1. 金在权的"功能还原"：涌现属性可还原但实现方式多样
2. 查尔默斯的"自然主义二元论"：意识是涌现的基本属性
3. 埃利斯等的"向下因果"：高层级具有真正的因果效力
4. 过程哲学的"事件本体论"：因果关系是过程内在的
```

前提1：如果认知过程X在外部世界进行，但与内部过程Y功能等同且耦合紧密
前提2：我们将内部过程Y视为认知
结论：我们应将外部过程X也视为认知的一部分

经典案例：奥托的笔记本
- 奥托患有阿兹海默症，使用笔记本记录重要信息
- 当他需要知道某个信息时，会查看笔记本
- 克拉克和查尔默斯认为：笔记本对奥托而言，等同于生物记忆
- 因此，认知过程延展到了笔记本这个外部工具

反对意见与辩护：
- 亚当斯和阿瓦：认知必须是因果过程，而笔记本缺乏正确的因果结构
- 梅纳瑞：认知是认知技能的应用，奥托使用笔记本的技能是认知的一部分
```

## 第四章：时间性与过程 —— 动态本体论而非静态存在

### 4.1 从静态存在到动态过程
- 4.1.1 巴门尼德的"存在"与赫拉克利特的"变化"
- 4.1.2 柏格森的"绵延"与创造进化论
- 4.1.3 怀特海的过程哲学：现实作为有机体的共生

**过程哲学的核心洞见**：```
怀特海的核心命题：

"过程即实在，实在即过程"
(Process is reality, reality is process)

替代亚里士多德的"实体-属性"框架：
- 不以"实体"作为基本存在
- 以"经验 occasion"作为基本单元
- 每个occasion是一个" drop of experience"

关键概念：
1. 共生(Concrescence)：多个occasion融合为一个新occasion的过程
2. 超体(Superject)：occasion完成后的客观存在状态
3. 摄入(Prehension)：occasion之间相互关联的方式
   - 积极摄入：对其他occasion的感受
   - 消极摄入：对其他occasion的排除
4. 永恒对象(Eternal Objects)：类似于柏拉图的理念，但存在于过程之中

对现代科学的意义：
- 量子力学：测量问题的过程解释
- 生物学：生命作为自创生过程
- 认知科学：意识作为过程而非实体
- 生态学：生态系统作为过程网络
```

### 4.2 时间的哲学：从线性时间到多重视域
- 4.2.1 现象学的时间意识：胡塞尔与海德格尔
- 4.2.2 时间的多重维度：物理时间、心理时间、社会时间
- 4.2.3 循环时间与线性时间的辩证：从古代宇宙论到现代性

**时间的多重维度分析**：```
时间的多维框架：

1. 物理时间 (Physical Time)
   - 特征：可测量、可量化、不可逆（热力学第二定律）
   - 度量：时钟时间、熵增方向
   - 哲学问题：时间是基本的还是涌现的？
   - 前沿：量子引力中的时间问题（圈量子引力、弦理论）

2. 心理时间 (Psychological Time)
   - 特征：主观体验、可压缩可扩展、可逆（记忆/想象）
   - 现象：时间的"流逝感"、"现在"的延展、记忆的重组
   - 神经机制：前额叶对时间序列的编码、海马体的情景记忆
   - 哲学问题：主观时间与客观时间的关系

3. 社会时间 (Social Time)
   - 特征：制度化、可协商、与权力相关
   - 形式：工作时间vs休闲时间、历法、仪式时间、截止日期
   - 批判：时间的商品化（"时间就是金钱"）
   - 变迁：从农业时间到工业时间再到后工业弹性时间

4. 生态时间 (Ecological Time)
   - 特征：多尺度、非线性、循环性
   - 尺度：地质时间、演替时间、季节循环、昼夜节律
   - 危机：人类时间尺度与生态时间尺度的错位（气候变化）
   - 智慧：传统生态知识中的时间智慧（如澳大利亚土著的"梦创时代"）

时间的辩证统一：
- 不同时间维度不是相互独立的，而是相互构成
- 社会实践协调不同时间维度（如工作日程协调生理节律与社会需求）
- 技术变革重塑时间体验（如即时通讯改变"等待"的含义）
- 时间政治：谁控制时间定义权，谁就掌握社会权力
```
